**Not sure or need flexibility?**

## Template Field Patterns

### Common Fields Across Templates

- **Feature Name** - What you're announcing/discussing
- **Problem/Benefit** - Why it matters to users
- **Description/Details** - The meat of your content
- **CTA/Link** - Where users should go next
- **Optional Context** - Pro tips, metrics, gratitude

### Input Best Practices

✅ **DO:**
- Be specific and concrete
- Include metrics when possible
- Write naturally (AI will polish it)
- Provide context for better results

❌ **DON'T:**
- Use vague descriptions
- Skip required fields
- Assume AI knows your product details
- Over-format (AI handles that)

## Platform Adaptations

### LinkedIn
- Professional tone
- Longer content (up to 3,000 characters)
- Use 3-5 hashtags
- Lead with value proposition

### X/Twitter
- Concise and punchy
- Thread format for longer content
- 2-3 hashtags max
- Strong hook in first line

### Instagram
- Visual-first approach
- Story-driven captions
- Use line breaks for readability
- Hashtags in comments

## Template Combinations

Mix and match for powerful content:

**Launch + How It Works**
→ Announce feature + guide users

**Problem → Solution + Before & After**
→ Show pain point + transformation

**Founder Voice + Feature Launch**
→ Personal story + product announcement

**Tips + Favorite Tool**
→ Actionable advice + tool recommendation
